## 프로젝트 정보

### 프로젝트 개발 환경

- IDE : Android Studio(Java) ,Visual Studio Code
- 기술 스택 : Node.js + Express, Mediapipe, YOLO, Tensorflow, Flask API
- 데이터베이스 : MySQL
- 서버 배포 : AWS EC2, AWS S3, AWS RDS
- 형상 관리 : Jira, Github, Notion
